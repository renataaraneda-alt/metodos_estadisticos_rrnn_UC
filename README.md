# metodos_estadisticos_rrnn_UC
Curso de estadística aplicado a estudios observacionales en el ámbito forestal y ambiental. Modelación con R: modelos lineales (GLM, mixtos), GAM, clasificación y aprendizaje supervisado/no supervisado, análisis multivariado y re-muestreo. Herramientas de Data Science y Machine Learning para explorar y sintetizar datos complejos del territorio.
