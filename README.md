Métodos Estadísticos para Recursos Naturales (2024)
¡Bienvenido/a! Este repositorio contiene los trabajos, análisis y códigos desarrollados durante el curso Métodos Estadísticos en Recursos Naturales, cursado en 2024 como parte de mi formación en Ingeniería Forestal.

El objetivo principal de este espacio es aplicar herramientas de modelación estadística avanzada y Machine Learning para explorar, sintetizar y analizar datos complejos del territorio y los recursos naturales, utilizando el software R.

🌲 Estructura del Repositorio
📁 Informes/: Contiene los reportes finales en formato PDF/HTML entregados al cierre de cada taller práctico.

📁 Scripts/: Códigos fuente en R utilizados para el procesamiento de datos, modelación y visualización.

⚠️ Nota de reproducibilidad: Los scripts se encuentran actualmente en una fase de revisión, depuración y optimización para asegurar que sean 100% replicables y sigan las mejores prácticas de reproducibilidad de la ciencia de datos.

📊 Contenidos y Herramientas Evaluadas
A lo largo del curso se abordaron metodologías orientadas a estudios observacionales (no experimentales) aplicadas a variables ambientales y forestales:

Modelos Lineales y Extensiones: Modelos de regresión, Modelos Mixtos y Modelos Lineales Generalizados (GLM).

Modelos No Paramétricos: Modelos Aditivos Generalizados (GAM) y otros enfoques predictivos.

Machine Learning y Clasificación: Métodos de clasificación supervisada y no supervisada.

Análisis Multivariado: Métodos clásicos de ordenación y reducción de dimensionalidad.
